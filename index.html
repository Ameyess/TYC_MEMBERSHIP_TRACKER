Discord Role Sync

Approved members get roles assigned automatically in their provincial + national server via the Railway bot.

How It Works

  1. Member submits Google Form
  2. Apps Script adds row to Sheet (Pending)
  3. Tracker loads it live — no manual import
  4. Admin clicks Approve
  5. Sheet updated → DiscordSynced = No
  6. Railway bot polls every 2 min
  7. Bot assigns role in provincial + national server
  8. Sheet updated → DiscordSynced = Yes
